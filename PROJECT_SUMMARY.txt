📦 PROJECT OVERVIEW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Together is a fully functional, production-ready Skribbl.io-style
multiplayer drawing game built with React, Node.js, and Socket.io.

✨ KEY FEATURES:
  ✅ Real-time multiplayer (2-8 players per room)
  ✅ Live drawing synchronization
  ✅ Word guessing with chat
  ✅ Score tracking & leaderboard
  ✅ 90-second timed rounds
  ✅ Modern responsive UI
  ✅ Zero database required
  ✅ Ready for production deployment

🎮 HOW TO PLAY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Create or join a room
2. Wait for 2+ players to join
3. Host clicks "Start Game"
4. Players take turns drawing
5. Others guess in chat
6. Score points for correct guesses
7. Play 3 rounds
8. Winner has most points!
